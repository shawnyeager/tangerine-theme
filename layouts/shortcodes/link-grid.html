{{/* Purpose: Renders a grid of links from Markdown content Parameters: - columns (string, optional): Number of columns - "1" or "2" (default: "2") Inner Content: - Markdown link list - Processed with markdownify Features: - Responsive grid layout - Single or two-column layouts - Automatically wraps content in grid card Usage: {{< link-grid >}} - [Link One](https://example.com) - [Link Two](https://example.com) {{< /link-grid >}} {{< link-grid columns="1" >}} - [Full Width Link](https://example.com) {{< /link-grid >}} Important Notes: - Inner content should be Markdown list of links - Two-column is default layout - Single-column adds "single-column" class for styling */}} {{- $columns := .Get "columns" | default "2" -}} {{- $class := "link-grid" -}} {{- if eq $columns "1" -}} {{- $class = "link-grid single-column" -}} {{- end -}}
{{- .Inner | markdownify -}}
